My Projects

YOLOv8 Object Detection app

During my first year as a student, I created a simple application using Gradio in Hugging Face, a Python library for building interactive machine learning interfaces. The application utilizes YOLOv8, a state-of-the-art detection model in 2022, to analyze uploaded images. It detects objects, draws bounding boxes around them, and classifies each object. This project served as an exciting entry point into machine learning and computer vision, giving me practical experience in integrating models and designing user-friendly interfaces.

DAA Project using Knapsack Algorithm

In my second year of college, as part of the Design and Analysis of Algorithms course, my team developed a mobile logistics application that optimizes truck loading using the knapsack algorithm. The app efficiently determines the best combination of items to fit onto pallets, maximizing both space utilization and profit. This project showcased my ability to apply algorithmic concepts to real-world problems, enhancing my skills in coding and problem-solving.

Kuwartrack: Hakbang Mo ay Laging Swak!

In my third year, my team and I created a simple financial tracker mobile app to help users manage their expenses effectively. The app features a Home page for sorting weekly expenses by category and viewing total spending, a Transaction page for tracking spending across time frames, setting budgets, and managing categories, and a Category Breakdown section for editing transactions and viewing detailed, read-only expense summaries. This project honed my skills in user-focused design and data management, delivering a practical tool for personal finance.
