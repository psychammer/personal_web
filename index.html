Featured Projects

YOLOv8 Object Detection Application

During my first year as a Computer Science student, I developed an interactive machine learning application using Gradio and Hugging Face. This project leverages YOLOv8, a cutting-edge object detection model, to analyze uploaded images, identify objects with high precision, and provide visual feedback through bounding boxes and classifications. This project marked my exciting entry into the world of AI and computer vision, giving me hands-on experience in model integration and user interface design.

Smart Logistics Optimizer (Knapsack Algorithm)

As part of my Design and Analysis of Algorithms course in second year, my team created a sophisticated mobile logistics application that revolutionizes truck loading efficiency. Using the knapsack algorithm, the app intelligently determines optimal item combinations for pallet loading, maximizing both space utilization and profit margins. This project demonstrated my ability to translate complex algorithmic concepts into practical, real-world solutions while enhancing my problem-solving and collaborative development skills.

Kuwartrack: Personal Finance Manager

In my third year, I led a team in developing "Kuwartrack: Hakbang Mo ay Laging Swak!" - a comprehensive financial tracking mobile application designed to empower users in managing their personal finances. The app features an intuitive Home dashboard for weekly expense categorization, a robust Transaction module for budget setting and expense tracking across multiple timeframes, and detailed Category Breakdown functionality for comprehensive financial analysis. This project strengthened my skills in user-centered design, database management, and creating practical solutions for everyday challenges.

CyberNix-12 Alphabot: Healthcare Companion for Isolated COVID-19 Patients

During my senior year of high school, I led the programming and hardware integration of “CyberNix-12 AlphaBot,” a healthcare companion robot designed to assist isolated COVID-19 patients. Using an Arduino Uno, I integrated multiple sensors, including heart rate, oximeter, and temperature sensors, into a crocheted robot that displayed vital signs and motivational messages via an LCD interface. The robot also featured medication reminders and a mobile UV sterilization cart (MACC) for hygiene. This project introduced me to embedded systems and healthcare tech, and gave me hands-on experience in rapid prototyping, hardware interfacing, and designing user-centric assistive technology.
